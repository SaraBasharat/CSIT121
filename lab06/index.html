Lab 06: UI/UX Evaluation

Website Evaluated: CDC (Centers for Disease Control and Prevention)

Website URL

https://www.cdc.gov

Website Name

Centers for Disease Control and Prevention (CDC)

Target Audience

This website is designed for healthcare professionals, patients, students, and the general public who want reliable health information.

Site Organization

The CDC website is organized with a clear top navigation bar that includes sections like Diseases & Conditions, Travel, and Data & Statistics. There is also a search bar and quick links on the homepage. This makes it easy to find information quickly.

CRAP Design Principles

  • Contrast: The site uses a blue and white color scheme, which makes the text and links stand out.
  • Repetition: The same navigation layout and buttons are used on every page.
  • Alignment: Content is cleanly aligned in columns and rows.
  • Proximity: Related information is grouped together, which makes the site easy to scan.

Accessibility Audit Score

Score: 59% — Not Compliant. The site has issues such as missing alt text and contrast problems.

Screenshot of Accessibility Checker

Site Effectiveness

The CDC site provides health information in a clear and organized way. The menus and search tools help users find what they need accurately.

Site Efficiency

The layout and search bar make it quick to find specific topics. It is well structured for fast navigation.

Engagement

The design is simple, clean, and professional. It fits the serious and trustworthy tone expected from a government health site.

Recommendation for Improvement

The website should improve accessibility by adding alt text to all images and fixing color contrast issues. This would help more users, including those with disabilities, access the site better.
